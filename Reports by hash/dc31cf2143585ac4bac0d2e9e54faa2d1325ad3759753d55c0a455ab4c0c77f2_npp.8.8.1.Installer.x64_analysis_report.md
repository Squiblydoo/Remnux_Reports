## 2. Classification

**Family**: Re-signed Notepad++ 8.8.1 installer — SmartScreen bypass  
**Confidence**: HIGH  
**Verdict**: NOT a malware payload; IS a deceptive/weaponized delivery wrapper

The outer NSIS executable (npp.8.8.1.Installer.x64.exe) has been stripped of its original signature and re-signed with a commercial EV certificate belonging to "Soft Insanity Oy" (a Finnish entity unrelated to Notepad++). The inner installer contents are consistent with the legitimate Notepad++ 8.8.1 release:

- 252 bundled files totaling 25.5 MB (LZMA compressed to 6.6 MB)
- Standard NSIS plugins: System.dll, nsDialogs.dll, InstallOptions.dll, LangDLL.dll, UserInfo.dll
- NPP core: notepad++.exe (legitimately signed by Don HO/Notepad++ team)
- NPP built-in plugins: NppExport.dll, mimeTools.dll, NppConverter.dll, nppPluginList.dll
- Updater: GUP.exe + libcurl.dll (legitimately signed)
- Context menu: NppShell.dll, NppShell.msix (legitimately signed)
- Localization, themes, config XML files — all standard NPP 8.8.1 assets

No additional executables, DLLs, scripts, or data files were found beyond the expected NPP 8.8.1 package.

## 3. Capabilities

- **SmartScreen bypass via EV certificate**: The NSIS outer wrapper carries a valid EV certificate from "Soft Insanity Oy," which suppresses the Windows SmartScreen "Unknown Publisher" dialog during execution, even though the cert holder has no relationship to Notepad++.
- **Privilege escalation via UAC**: NSIS installer requests administrator rights to write to `%ProgramFiles%\Notepad++`. peframe confirms `ElevatePrivileges` behavior; ANY.RUN confirms `privEscalation: true`.
- **Shell extension registration**: Registers `NppShell.dll` via `regsvr32 /s` to provide right-click context menu integration — standard NPP behavior.
- **Update check**: Installed Notepad++ contacts `notepad-plus-plus.org/update/getDownloadUrl.php?version=8.81&param=x64` on first launch — standard NPP auto-update.
- **Self-deletion**: NSIS drop to `%TEMP%` with temp filename pattern; standard NSIS behavior.

## 4. Attack Chain

```
User downloads re-signed installer (believing it's from Don HO)
       ↓
EV cert from "Soft Insanity Oy" suppresses SmartScreen warning
       ↓
NSIS stub runs → unpacks LZMA payload to %TEMP%/$PLUGINSDIR
       ↓
Installs legitimate Notepad++ 8.8.1 to %ProgramFiles%\Notepad++
       ↓
Registers NppShell.dll via regsvr32 (context menu)
       ↓
Launches notepad++.exe via Explorer (proxy execution)
       ↓
NPP runs normally — no malicious post-install behavior
```

**Threat**: The attack is positional — the risk is that a user trusts the installer's EV signature and installs what appears to be a verified-publisher product, potentially as part of a phishing campaign, supply-chain impersonation, or SEO-poisoning attack. The legitimate NPP binary provides no additional security risk once installed, but the delivery mechanism circumvents OS trust controls.

## 8. Analyst Notes

### Assessment
This is a **re-signed legitimate Notepad++ 8.8.1 installer**. The threat is positional: the EV certificate from "Soft Insanity Oy" bypasses Windows SmartScreen's publisher-reputation check while presenting a seemingly authoritative green-padlock trust indicator to users who inspect the signature. The underlying installation is benign.

### Uncertainty
The inner `notepad++.exe` hash (`cca6091dce76e65bdd610cf1521b8e85b6aad8144aa629ddb11613183e3630e6`) has not been independently verified against the official NPP 8.8.1 GitHub release artifact. The legitimate DigiCert signature from the "Notepad++" entity (serial `03aa6492de9d96a90a4bca97beadb44a`) is present and would require forging the CA to spoof. **Verification recommended**: compare against the official `npp.8.8.1.Installer.x64.exe` SHA256 from `github.com/notepad-plus-plus/notepad-plus-plus/releases/tag/v8.8.1`.

### Possible Delivery Context
This sample is likely distributed via:
- SEO-poisoned search results ("notepad++ download")
- Phishing sites impersonating notepad-plus-plus.org
- Malvertising or typosquat domains

### Recommended Follow-up
1. Compare inner `notepad++.exe` hash against official NPP 8.8.1 GitHub release
2. Investigate "Soft Insanity Oy" — check for other re-signed binaries using serial `0db0cb5c25bc7af916078b05a5902aed`
3. Identify distribution infrastructure — the original download source
4. Submit certificate serial to CA (GoGetSSL/EnVers Group) for revocation
