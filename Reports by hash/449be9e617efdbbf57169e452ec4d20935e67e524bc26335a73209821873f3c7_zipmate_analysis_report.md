## Summary

Zipmate.exe is **adware / browser hijacker** disguised as a "ZipMate" file archiving utility. It presents a multi-page WPF installer UI to the user, then silently manipulates the Firefox browser via automated Win32 keyboard injection to install a search engine extension and change the user's default search engine. This is a classic **pay-per-install search hijacking** operation, with full affiliate campaign tracking reported back to a C2 server.
