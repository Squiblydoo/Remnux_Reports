## 4. Capabilities

### Authentication & Fingerprinting
- Registers as custom URL protocol handler: `fivemods://` (`app.setAsDefaultProtocolClient`)
- Discord OAuth flow: opens `https://fivemods[.]app/api/authorize` in browser; local HTTP server on **port 4311** receives callback with bearer token
- Token stored locally and sent as `Authorization` header on all API calls
- **System fingerprint** collected by `get-system-info` (sent to server):
  - CPU: manufacturer, brand, speed
  - RAM: total GB
  - GPU: model and VRAM
  - Display: primary screen resolution (width Ă— height)

### FiveM Game File Manipulation ("Mods Engine")
- Locates FiveM installation via `HKCU\SOFTWARE\CitizenFX\FiveM\Last Run Location`
- Detects game store type (Steam: `steam_api64.dll`; Epic: `.egstore`; MS Store: `appxmanifest.xml`)
- Downloads signed RPF archives from C2 CDN
- Installs mods as `+update.rpf_*` files in the FiveM game storage path
- A local WebSocket-style IPC channel (`H()`) coordinates between the Electron UI and the "mods engine" component with `engine_mode` messages: `upload`, `complete_installation`, `duplicate_update_store`
- Process killing via `fkill`/`find-process` before file operations (kills FiveM while modifying files)
- File timestamp modification via `utimes` module (anti-forensics potential)
- PowerShell execution: checks admin status via `[Security.Principal.WindowsPrincipal] [Security.Principal.WindowsIdentity]::GetCurrent()`
- WScript.Shell execution for additional command dispatch

### Persistence (Two Mechanisms)
**1. Windows Scheduled Task** (created on "client mode" enable):
```
PowerShell: Register-ScheduledTask -Action (New-ScheduledTaskAction -Execute "<execPath>" "<args>") 
            -Trigger (New-ScheduledTaskTrigger -Daily -At '18:00:00') -TaskName <taskName>
Fallback: schtasks /Create /SC DAILY /TN "<name>" /TR "<path> --task" /ST 18:00:00
```
- Task runs daily at 18:00 with `--task` argument (headless/no-UI mode)

**2. Login Item** (created on "client mode" enable):
```javascript
app.setLoginItemSettings({
  openAtLogin: true,
  openAsHidden: true,  // hidden â€” no taskbar/tray icon at login
  name: "ModsHub",
  args: ["--task"]
});
```
- Starts silently at every Windows login; `--task` flag suppresses main window

### Subscription & Account Management
- Subscription tiers: `fivemods_plus` and `fivemods_cars` (measured in days remaining)
- Subscription gates feature availability; expired subscriptions disable "client mode"
- Registry key migration from `FiveMods` â†’ `ModsHub` branding (`HKCU\Software\FiveMods` â†’ `HKCU\Software\ModsHub`)

### Auto-Update
- `electron-updater` polls `https://s0.fivemods[.]app/app` (production) for app updates
- Dev environment uses `https://dev.s0.fivemods[.]io/app`

## 10. Analyst Notes

### What This Is
ModsHub is a subscription-based FiveM cheating/modification service delivered via an Electron app. It provides paying subscribers with unauthorized game mods (likely including custom vehicles, ESP, or ESX-based advantage mods) by directly injecting RPF archives into FiveM's game data, bypassing FiveM's content verification. This is ToS violation software, not a traditional infostealer.

### What This Is NOT
- **Not a credential stealer**: No browser `Login Data`, `Cookies`, or `Local State` file access found
- **Not a Discord token stealer**: Uses proper OAuth 2.0 flow (standard browser-based, port 4311 callback)
- **Not a crypto wallet stealer**: No seed phrase, private key, or wallet extension enumeration
- **Not ransomware**: `ValuableFileExtensions` YARA match is a false positive

### Key Risk Factors
1. **Hidden persistent execution**: Daily scheduled task + hidden login item run without user interaction or visible UI
2. **System fingerprinting**: CPU/GPU/RAM/display sent to remote server on every session
3. **Individual developer cert**: Low barrier to obtain; certificate holder identity unverified beyond name/location
4. **FiveM ToS violation**: Using this software risks ban from FiveM servers and the Rockstar Social Club
5. **Obfuscation**: JavaScript-Obfuscator string-array shuffling with 1,573-element string table; anti-devtools flags (`devTools: false`) on all windows
6. **Process killing**: Force-kills processes (likely FiveM) before game file manipulation

### Recommended Follow-Up
- Behavioral sandbox run on extracted `ModsHub.exe` with a network intercept proxy (Burp/mitmproxy) to capture the full API call sequence and any undocumented endpoints
- Review `fivemods.io` / `fivemods.app` WHOIS and ASN history for attribution
- Check if the `extraHosts[]` array contains additional domains (requires full JS deobfuscation with controlled Node.js execution)
- Examine the "mods engine" component more deeply â€” it likely involves a second PE (DLL or EXE) that performs the actual FiveM patching at a lower level than this JavaScript shows
