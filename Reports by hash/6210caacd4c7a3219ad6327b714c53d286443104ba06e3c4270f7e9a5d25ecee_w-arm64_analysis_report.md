## 3. Capabilities

### C2 Communication
- Polls `https://webhook902.securitysolut[.]com` over HTTPS
- Check-in authentication via **HMAC-SHA256(secret_key, unix_timestamp)** â†’ hex-encoded token

## 9. Analyst Notes

**ARM64 targeting**: This binary specifically targets Windows ARM64 â€” a notably uncommon delivery target for malware. This suggests either a targeted deployment against ARM64 endpoints (Qualcomm Snapdragon X Elite laptops, Surface Pro X) or an attacker building for a wider ARM64 foothold they anticipate becoming more prevalent.

**Sectigo EV cert from Pingxiang, Jiangxi CN**: Jiangxi-based entity; differs from the Xiamen (Fujian) / Weihai (Shandong) pattern seen in previous samples â€” no shared cert serial or C2 to link this to prior campaigns under the cross-reference policy.

**`securitysolut[.]com` domain**: The domain name mimics a security services vendor (`security solution`) â€” a common social engineering technique for C2 infrastructure. The `webhook902` subdomain suggests automated/programmatic generation or a numerically indexed deployment.

**HMAC auth with timestamp prevents replay**: The HMAC-SHA256(secret, timestamp) scheme means captured traffic cannot be replayed; the token is only valid for the current second/window. The binary won't check in without the correct secret.

**`github.com/custom-socks5` is private**: The module path does not resolve to a public GitHub repository (as of writing), suggesting either a private repo or an entirely fabricated module path used for misdirection.

**Self-destruct on max retries**: The agent self-destructs autonomously if the C2 is unreachable after a configured number of retries â€” unusual for commodity malware, more consistent with an operator-controlled, short-lived implant designed to clean up after itself.

**Recommended follow-up**:
- Block `securitysolut[.]com` and all subdomains at DNS level
- Search endpoint fleet for ARM64 Windows processes with the Go build fingerprint (`go1.26.3`, `custom-socks5`)
- Monitor `%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\*.lnk` for unauthorized shortcuts
- Revocation request to Sectigo for cert serial `6c5efe09cd24511fddd320dd409c2d03`
- Pivot on HMAC secret `b9b264513a4f9074dc8fa9c022fce263` across network logs if full C2 traffic is captured
