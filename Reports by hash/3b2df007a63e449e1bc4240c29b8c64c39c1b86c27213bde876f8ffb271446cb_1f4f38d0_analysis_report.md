## 2. Classification

- **Family**: ZEC Wallet Crypto-Lure / Remote Utilities RAT Dropper
- **Confidence**: HIGH
- **Category**: Remote Access Trojan (RAT) deployer, abusing a legitimate commercial remote admin tool

**Reasoning**: The MSI presents a fully functional Zecwallet Fullnode Electron app as a decoy while silently downloading and installing Remote Utilities Host (`rutserv.exe` + `rfusclient.exe`) from a malicious C2 domain (`zupdate.icu`). A PowerShell post-install script systematically hides the Remote Utilities installation using `attrib +s +h`, removes it from Add/Remove Programs, and hides follow-on persistence scripts. This is a well-known technique used by threat actors targeting cryptocurrency users: deliver a real wallet app to reduce suspicion while implanting remote access.

---

## 3. Capabilities

- **Decoy installation**: Installs a fully functional Zecwallet Fullnode v1.8.7.0 (Electron-based Zcash wallet with `zcashd.exe` v5.4.2.50, `zcashcli.exe`, `app.asar`, ~350MB of legitimate files)
- **Remote access**: Downloads and silently installs Remote Utilities Host from `http://zupdate[.]icu/se.msi` (labeled "ZEC Block Data")
- **Persistence**: Downloads and executes 10 sequentially numbered `.bat` scripts from `zupdate.icu` (contents unknown without sandbox execution; names suggest display/stealth/security-center manipulation/persistence setup)
- **Defense evasion**: Hides Remote Utilities folder and executables with `attrib +s +h` (system + hidden attributes)
- **Visibility suppression**: Sets `HKCU\...\Explorer\Advanced\ShowSuperHidden=2` to prevent "show hidden files" from revealing the hidden RAT
- **Uninstall concealment**: Sets `SystemComponent=1` in the Remote Utilities uninstall key — removes it from Add/Remove Programs in Control Panel
- **Persistence script staging**: Hides `C:\Users\Public\update.ps1` and `C:\Users\Public\update7.ps1` (likely the update/beacon bat scripts staged by the prereq chain)
- **Sandbox/VM detection**: Checks for VMware, VirtualBox (YARA: `BlacklistSandbox`), enumerates processes
- **WMI recon**: Multiple WMI queries for OS/hardware fingerprinting
- **PowerShell execution**: Uses `Binary.PowerShellScriptLauncher.dll` (Advanced Installer component) to run inline PS1 at sequence 6402

## 4. Attack Chain

```
1. Victim downloads/runs 1f4f38d0.msi (ZEC Wallet installer)
        ↓
2. MSI installs legitimate Zecwallet Fullnode to %APPDIR%
   (decoy: real zcashd.exe, app.asar, locale paks)
        ↓
3. AI_PreRequisite chainer (AI_LaunchChainer seq 6598) downloads from http://zupdate[.]icu/:
   a. 1display.bat        ← seq 0
   b. setup3.bat          ← seq 2
   c. 3.1setuphd.bat      ← seq 3
   d. 4h.bat              ← seq 4  (likely: hide files)
   e. 5pause.bat          ← seq 5
   f. 6last.bat           ← seq 6
   g. 7SecurityCenter.bat ← seq 7  (likely: disable Windows Defender/Security Center)
   h. 8display2.bat       ← seq 8
   i. 9last.bat           ← seq 9
   j. 10setup2.bat        ← seq 10
   k. se.msi              ← "ZEC Block Data" — silently installs Remote Utilities Host
                            (only if GUID {CA01AB2D-E912-4FC0-AD52-2D610BE0D1CF} absent)
        ↓
4. PowerShellScriptInline (seq 6402, via PowerShellScriptLauncher.dll):
   - attrib +s +h "C:\Program Files (x86)\Remote Utilities - Host"
   - attrib +s +h rutserv.exe + rfusclient.exe
   - attrib +s +h "C:\ProgramData\Remote Utilities"
   - attrib +s +h C:\Users\Public\update.ps1 + update7.ps1
   - REG: ShowSuperHidden=2 (block hidden-file reveal in Explorer)
   - REG: SystemComponent=1 (hide from Add/Remove Programs)
   - Stop-Process explorer (force-reload settings)
        ↓
5. Remote Utilities Host running as hidden service with attacker's Internet-ID
   → Full persistent remote desktop / RCE access to victim machine
```

**Note on Remote Utilities**: Remote Utilities (famatech.com) is a legitimate commercial RMM tool. When configured with an attacker-controlled Internet-ID, it establishes outbound reverse connections through Famatech's relay servers, bypassing firewalls and NAT. The attacker needs only their own Remote Utilities Viewer and the preconfigured Internet-ID to achieve persistent full remote access. The bat scripts (contents not recovered) likely configure this Internet-ID and set up further persistence.

## 9. Analyst Notes

**Gaps / recommended follow-up:**
1. **Bat script contents not recovered**: The 10 .bat scripts are downloaded live from `zupdate.icu` and not embedded in the MSI. Their full content (Internet-ID configuration, persistence mechanism, Defender disabling commands) is the most critical unknown. Try pulling them with `curl http://zupdate.icu/7SecurityCenter.bat` etc. from a sandboxed environment.
2. **se.msi not analyzed**: The Remote Utilities payload MSI from `http://zupdate.icu/se.msi` is not embedded — it's the actual Remote Utilities distribution with the attacker's Internet-ID preconfigured. Capturing and analyzing this would reveal the attacker's Famatech relay credentials.
3. **Targeting**: ZEC Wallet (Zcash) is a privacy-focused cryptocurrency. Targeting users of this wallet suggests the threat actor is interested in financial assets and/or the privacy-conscious demographic. This may be part of a broader crypto-targeting campaign.
4. **Cert reuse tracking**: The Sectigo EV cert for "Wuxi ENJOY International Trading Co., Ltd." (serial `00e8141cb236b6b9bb3dfe5d94f345ca16`) should be checked against other samples for infrastructure reuse.
5. **Tria.ge retry**: Submit from a different IP or via web UI at https://tria.ge to obtain full dynamic behavioral analysis.
